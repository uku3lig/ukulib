# Various utilities

This page serves as a non-exhaustive list of utilities provided by ukulib.

## `Ukutils`

[`Ukutils`](https://maven.uku3lig.net/javadoc/releases/net/uku3lig/ukulib-common/latest/.cache/unpack/net/uku3lig/ukulib/utils/Ukutils.html) is a beautiful utility class with a bunch of functions that may or may not be of use to you. [`PlatformUkutils`](https://maven.uku3lig.net/javadoc/releases/net/uku3lig/ukulib-common/latest/.cache/unpack/net/uku3lig/ukulib/utils/PlatformUkutils.html) also provides a few methods that have differing implementations depending on the mod loader.

## `SilentButtonWidget`

[`SilentButtonWidget`](https://maven.uku3lig.net/javadoc/releases/net/uku3lig/ukulib-common/latest/.cache/unpack/net/uku3lig/ukulib/utils/SilentButtonWidget.html) is virtually identical to a button, but without the sound when it's clicked. Useful if you want to make a button that plays another sound.

## `PlayerArgumentType` :fabric:

[`PlayerArgumentType`](https://maven.uku3lig.net/javadoc/releases/net/uku3lig/ukulib-fabric/latest/.cache/unpack/net/uku3lig/ukulib/fabric/PlayerArgumentType.html) is a _client_ argument type that can be used to autocomplete to any player present in the current world, but also allows to take in an UUID. See [the Fabric documentation](https://docs.fabricmc.net/develop/commands/basics) for usage.
