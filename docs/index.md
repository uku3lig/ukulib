**ukulib** is a Minecraft library mod developed throughout the years for my multitude of mods, with many useful capabilities!

!!! warning

    This documentation only concerns ukulib 2.0.0 and later, and therefore only covers Minecraft 26.1 and later.

## Features

- Compatible with both Fabric and NeoForge
- Simple but comprehensive and powerful configuration system, with utilities and widgets for config screens
- Various Minecraft related utilities
- ... and more to come!
